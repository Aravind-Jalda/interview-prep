# DevOps Security Scenario-Based Questions & Answers (DevOps - 2 to 5 Years)

---

## 1. Credentials are exposed in code. What will you do?

- Remove credentials  
- Rotate keys  
- Use secret manager  

---

## 2. Unauthorized access detected. What will you check?

- IAM roles  
- Access logs  
- Permissions  

---

## 3. How do you secure CI/CD pipeline?

- Use secret manager  
- Avoid hardcoding  
- Restrict access  

---

## 4. How do you secure Docker images?

- Use minimal base image  
- Scan images  
- Avoid root user  

---

## 5. How do you secure Kubernetes cluster?

- Use RBAC  
- Use network policies  
- Use secrets  

---

## 6. SSL is not working. What will you check?

- Certificate validity  
- Port 443  
- Load balancer config  

---

## 7. How do you manage secrets in Kubernetes?

- Use Secrets

- Monitor logs

- Update dependencies
