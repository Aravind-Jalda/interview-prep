# Networking Interview Questions & Answers (DevOps - 2 to 5 Years)

---

## 1. What is a network?
A network is a group of connected devices that can communicate with each other.  
In DevOps, it connects servers, applications, and services.

---

## 2. What is an IP address?
An IP address is a unique identifier assigned to a device in a network.  
Example: 192.168.1.1

---

## 3. What is the difference between public and private IP?
- Public IP → accessible from internet  
- Private IP → used inside internal network  

---

## 4. What is DNS?
DNS (Domain Name System) converts domain names into IP addresses.  
Example: google.com → IP address  

---

## 5. What is a port?
A port is a communication endpoint.  
Used to identify services on a server.

Examples:
- 22 → SSH  
- 80 → HTTP  
- 443 → HTTPS  

---

## 6. What is HTTP vs HTTPS?
- HTTP → not secure  
- HTTPS → secure using SSL/TLS encryption  

---

## 7. What is TCP vs UDP?
- TCP → reliable, connection-oriented  
- UDP → faster, connectionless  

---

## 8. What is a subnet?
A subnet divides a network into smaller parts.  
Used for better organization and security.

---

## 9. What is a gateway?
A gateway connects one network to another.  
Example: router connecting your network to internet.

---

## 10. What is NAT?
NAT (Network Address Translation) allows private IPs to access the internet using a public IP.

---

## 11. What is a firewall?
A firewall controls incoming and outgoing traffic based on rules.  
Example: security groups, NSG.

---

## 12. What is load balancing?
Load balancing distributes traffic across multiple servers.  
Improves performance and availability.

---

## 13. What is ping?
`ping` checks connectivity between two systems.

Example:
ping google.com

---

## 14. What is traceroute?
`traceroute` shows the path taken by packets to reach destination.

---

## 15. What is netstat?
`netstat` shows active connections and listening ports.

---

## 16. What is curl?
`curl` is used to send requests to a server and check responses.

---

## 17. What is SSH?
SSH (Secure Shell) is used to connect to remote servers securely.

---

## 18. What is bandwidth?
Bandwidth is the amount of data that can be transferred over a network.

---

## 19. What is latency?
Latency is the time taken for data to travel from source to destination.

---

## 20. What is CDN?
CDN (Content Delivery Network) stores content closer to users for faster delivery.

---

## Bonus: Networking in DevOps

Used for:
- Connecting services
- Debugging issues
- Securing applications
- Cloud networking (VPC, VNet)
