# Grafana Scenario-Based Interview Questions & Answers (DevOps - 2 to 5 Years)

---

## 1. Dashboard is not showing data. What will you check?

- Check data source connection  
- Verify query  
- Check time range  
- Ensure Prometheus is working  

---

## 2. Graph is empty. Why?

- Wrong query  
- No data in selected time range  
- Metrics not collected  

---

## 3. Alerts are not working. What will you check?

- Alert condition  
- Notification channel  
- Data source  
- Threshold configuration  

---

## 4. How do you create a dashboard?

- Add data source  
- Create panels  
- Write queries  
- Save dashboard  

---

## 5. How do you debug query issues?

- Use Explore mode  
- Test PromQL queries  
- Check metrics  

---

## 6. How do you monitor CPU usage?

- Use Prometheus metric  
- Create graph panel  
- Set threshold  

---

## 7. How do you reduce dashboard load time?

- Optimize queries  
- Reduce time range  
- Use efficient panels  

---

## 8. How do you manage multiple environments?

- Use variables  
- Create dynamic dashboards  

---

## 9. How do you handle alert noise?

- Adjust thresholds  
- Use alert grouping  
- Reduce duplicate alerts  

---

## 10. How do you share dashboards?

- Export JSON  
- Share link  
- Use Grafana provisioning  

---

## 11. How do you monitor Kubernetes using Grafana?

- Connect Prometheus  
- Import dashboards  
- Visualize pod metrics  

---

## 12. How do you monitor logs in Grafana?

- Use Loki  
- Query logs  
- Create dashboards  

---

## 13. How do you secure Grafana?

- Use authentication  
- Role-based access  
- HTTPS  

---

## 14. How do you backup dashboards?

- Export JSON  
- Use version control  

---

## 15. How do you customize dashboards?

- Use variables  
- Change panels  
- Add filters  

---

## 16. How do you integrate Grafana with alert tools?

- Configure notification channels  
- Connect Slack/email  

---

## 17. What will you do if Grafana is slow?

- Optimize queries  
- Reduce panels  
- Increase resources  

---

## 18. How do you monitor application performance?

- Track latency  
- Track request count  
- Visualize metrics  

---

## 19. How do you troubleshoot missing metrics?

- Check Prometheus  
- Check query  
- Check labels  

---

## 20. Explain real-time Grafana usage

Example:

"I used Grafana with Prometheus to create dashboards for CPU, memory, and application metrics, and configured alerts for failures."
