# Ansible Scenario-Based Interview Questions & Answers (DevOps - 2 to 5 Years)

---

## 1. Playbook failed. What will you do?

- Check error message  
- Verify YAML syntax  
- Check logs  
- Fix and rerun  

---

## 2. Unable to connect to server. What will you check?

- SSH connectivity  
- Inventory file  
- Credentials  
- Firewall  

---

## 3. How do you deploy application using Ansible?

- Write playbook  
- Define tasks  
- Run playbook  

---

## 4. How do you manage multiple servers?

- Run playbook

## 5. How do you reuse code?

- Use roles

## 6. How do you handle configuration changes?

- Update playbook
