# Azure Scenario-Based Interview Questions & Answers (DevOps - 2 to 5 Years)

---

## 1. Your application hosted on Azure is down. What will you check?

- VM/App Service status  
- NSG rules  
- Load balancer health  
- Logs (Azure Monitor)  
- DNS configuration  

---

## 2. VM is not accessible via SSH/RDP. What will you check?

- NSG rules (port 22/3389)  
- Public IP assigned  
- VM status  
- Firewall rules  

---

## 3. How do you design high availability in Azure?

- Use Availability Sets or Zones  
- Use Load Balancer  
- Use multiple VMs  
- Use managed services  

---

## 4. How do you secure Azure environment?

- Use AAD for authentication  
- Enable MFA  
- Use NSG rules  
- Use Managed Identity  
- Restrict public access  

---

## 5. How do you deploy application in Azure?

- Use VM or App Service  
- Use Azure Pipeline for CI/CD  
- Configure networking  
- Monitor application  

---

## 6. How do you reduce Azure cost?

- Stop unused VMs  
- Use reserved instances  
- Optimize storage  
- Monitor usage  

---

## 7. How do you allow Azure VM to access storage?

- Use Managed Identity  
- Assign required permissions  

---

## 8. How do you monitor Azure resources?

- Use Azure Monitor  
- Set alerts  
- Use Log Analytics  

---

## 9. How do you handle traffic spike?

- Scale VMs (VM Scale Sets)  
- Use Load Balancer  
- Use auto scaling  

---

## 10. How do you handle disk full issue?

- Check disk usage  
- Clean logs  
- Resize disk  
- Add new disk  

---

## 11. How do you manage secrets in Azure?

- Use Azure Key Vault  
- Avoid hardcoding  
- Use secure access  

---

## 12. How do you integrate Azure DevOps with Git?

- Use Azure Repos or GitHub  
- Configure pipeline triggers  
- Automate build and deployment  

---

## 13. How do you deploy containerized apps?

- Use AKS  
- Build Docker image  
- Push to registry  
- Deploy using kubectl  

---

## 14. How do you backup data in Azure?

- Use Azure Backup  
- Enable snapshots  
- Store in Blob Storage  

---

## 15. How do you restrict production access?

- Use RBAC  
- Limit permissions  
- Use approval process  

---

## 16. How do you connect private network to internet?

- Use NAT Gateway  
- Configure routing  

---

## 17. What will you do if logs are missing?

- Check logging configuration  
- Verify Azure Monitor  
- Check permissions  

---

## 18. How do you migrate app to Azure?

- Lift and shift (VM)  
- Use App Service  
- Test and validate  

---

## 19. How do you troubleshoot slow application?

- Check CPU/memory  
- Analyze logs  
- Scale resources  
- Optimize code  

---

## 20. Explain a real-time Azure project

Example:

"I worked on deploying an application in Azure where:
- VMs were used for hosting  
- Load balancer handled traffic  
- Azure DevOps pipeline automated deployment  
- Azure Monitor tracked performance"
