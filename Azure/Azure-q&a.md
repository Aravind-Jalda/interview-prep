# Azure Interview Questions & Answers (DevOps - 2 to 5 Years)

---

## 1. What is Azure?
Azure is Microsoft’s cloud platform that provides services like compute, storage, networking, and databases.

---

## 2. What is a Virtual Machine (VM)?
Azure VM is similar to AWS EC2.  
It is a virtual server used to run applications.

---

## 3. What is Azure Storage?
Azure Storage provides services like:
- Blob Storage → for files  
- Disk Storage → for VMs  
- File Storage → shared file system  

---

## 4. What is Blob Storage?
Blob Storage is used to store unstructured data like images, videos, logs, and backups.

---

## 5. What is a Resource Group?
A Resource Group is a container that holds related Azure resources.  
It helps manage and organize resources together.

---

## 6. What is Azure Virtual Network (VNet)?
VNet is a private network in Azure.  
It is similar to AWS VPC.

---

## 7. What is a Subnet in Azure?
Subnet is a smaller network inside a VNet.  
Used to separate resources for security and management.

---

## 8. What is Network Security Group (NSG)?
NSG is a firewall that controls inbound and outbound traffic for VMs and subnets.

---

## 9. What is Azure Load Balancer?
It distributes traffic across multiple VMs to improve availability.

---

## 10. What is Azure App Service?
App Service is a managed platform to deploy web applications without managing servers.

---

## 11. What is Azure Kubernetes Service (AKS)?
AKS is a managed Kubernetes service used to run containerized applications.

---

## 12. What is Azure DevOps?
Azure DevOps is a platform that provides:
- Repos (Git)
- Pipelines (CI/CD)
- Boards (tracking)

---

## 13. What is Azure Pipeline?
Azure Pipeline is used to automate build, test, and deployment.

---

## 14. What is Azure Active Directory (AAD)?
AAD is used for identity and access management in Azure.

---

## 15. What is a Managed Identity?
Managed Identity allows Azure services to access resources securely without storing credentials.

---

## 16. What is Azure Monitor?
Azure Monitor is used to collect metrics, logs, and alerts.

---

## 17. What is Azure Log Analytics?
It is used to analyze logs and monitor systems.

---

## 18. What is Azure SQL Database?
A managed relational database service similar to AWS RDS.

---

## 19. What is Availability Set?
Ensures VMs are distributed across fault domains to improve availability.

---

## 20. What is Infrastructure as Code in Azure?
Using tools like ARM templates, Bicep, or Terraform to provision resources.

---

## Bonus: How Azure is used in DevOps?

- CI/CD using Azure Pipelines  
- Hosting apps (VM, App Service, AKS)  
- Monitoring using Azure Monitor  
- Managing infrastructure using IaC
