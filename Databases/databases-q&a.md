# Database Interview Questions & Answers (DevOps - 2 to 5 Years)

---

## 1. What is a database?
A database is a system used to store and manage data.

---

## 2. What is MySQL?
MySQL is an open-source relational database used to store structured data.

---

## 3. What is RDBMS?
Relational Database Management System stores data in tables with relationships.

---

## 4. What is SQL?
SQL is a language used to interact with databases.

---

## 5. What are basic SQL operations?

- SELECT → read data  
- INSERT → add data  
- UPDATE → modify data  
- DELETE → remove data  

---

## 6. What is a table?
A table stores data in rows and columns.

---

## 7. What is primary key?
Unique identifier for each record.

---

## 8. What is index?
Used to improve query performance.

---

## 9. What is normalization?
Organizing data to reduce redundancy.

---

## 10. What is transaction?
A group of operations executed together.

---

## 11. What is ACID property?

- Atomicity  
- Consistency  
- Isolation  
- Durability  

---

## 12. What is database backup?
Copy of database for recovery.

---

## 13. What is replication?
Copying data from one database to another.

---

## 14. What is scaling in database?

- Vertical → increase resources  
- Horizontal → add more servers  

---

## 15. What is connection string?
Used by application to connect to database.

---

## 16. What is database latency?
Time taken for query execution.

---

## 17. What is read replica?
Used to handle read traffic.

---

## 18. What is caching?
Store frequently accessed data for faster response.

---

## 19. How databases are used in DevOps?

- Application backend  
- Data storage  
- Performance tuning  
- Backup and recovery  

---

## 20. Example flow

Application → Database → Query → Response
