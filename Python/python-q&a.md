# Python Interview Questions & Answers (DevOps - 2 to 5 Years)

---

## 1. Why Python is used in DevOps?
Python is simple and powerful for automation.  
It is used for scripting, API calls, file handling, and automation tasks.

---

## 2. What are Python features?
- Easy to learn  
- Cross-platform  
- Large library support  
- Good for automation  

---

## 3. What is a variable in Python?
A variable stores data.

Example:
x = 10

---

## 4. What are data types in Python?
Common types:
- int  
- float  
- string  
- list  
- dictionary  

---

## 5. What is a list?
A list is a collection of items.

Example:
[1, 2, 3]

---

## 6. What is a dictionary?
Stores key-value pairs.

Example:
{"name": "devops"}

---

## 7. What is a function?
A function is reusable code.

Example:
def add(a, b):
    return a + b

---

## 8. What is a loop?
Used to repeat tasks.

Example:
for i in range(5):
    print(i)

---

## 9. What is a conditional statement?
Used to make decisions.

Example:
if x > 10:
    print("High")

---

## 10. What is a module?
A file containing Python code that can be reused.

---

## 11. What is pip?
pip is a package manager used to install Python libraries.

---

## 12. What is virtual environment?
Used to isolate Python dependencies.

---

## 13. What is exception handling?
Used to handle errors.

Example:
try:
    x = 10/0
except:
    print("Error")

---

## 14. What is file handling?
Reading/writing files.

Example:
open("file.txt")

---

## 15. What is JSON in Python?
Used to handle API data.

---

## 16. What is requests library?
Used to call APIs.

Example:
import requests

---

## 17. What is subprocess module?
Used to run shell commands from Python.

---

## 18. What is logging in Python?
Used to record logs instead of print statements.

---

## 19. How Python is used in DevOps?

- Automation scripts  
- API integration  
- Log parsing  
- CI/CD tasks  

---

Script to check server health and send alert
